PidControllerクラス説明書

PID制御を実装したクラスです。
doubleで実装。doubleの演算回路を載せたマイコン向けです。

  • 使用に当たって準備すること
    1. #include "your_using_der/PidController.hpp" と書いてインクルードする。
    2. PidController.cpp に本体が書いてあるので hpp ファイルと同じディレクトリに配置する。
    3. このクラスは ctrl ネームスペース内に配置されているのでクラス呼び出し時に
      ctrl:: をつけましょう。

1. PidController()

コンストラクタです. 初期化に使います

番号 引数名 型 説明
第1引数 kp double P(比例)成分の係数
第2引数 ki double I(積分)成分の係数
第3引数 kd double D(微分)成分の係数
第4引数 max_output_value double 出力の最大値を指定
第5引数 attach_period_s double (オプション)メンバ関数calculateの割り込み周期. 単位は秒で指定
---— --------------— -----— ----—
返り値 xxxx void 返り値なし

2. calculate()

目標値と現在値から適切な値を出力するPID制御の心臓部です. これが使えれば終わりです

番号 引数名 型 説明
第1引数 targetValue double 目標値(目標の速度とか)
第2引数 inputValue double 現在値(現在の速度とか)
第5引数 attach_period_s double (オプション)時間割り込みを使わないときに前回PIDしたときからの時間を指定できる. 単位は秒
---— --------------— -----— ----—
返り値 PID_output double PIDにより決定された値

3. setCoefficient

PIDの各係数をセットします

番号 引数名 型 説明
第1引数 coefficient double 係数の値
第2引数 selector double 0でkp, 1でki, 2でkd成分を選択. KP,KI,KDの定数もあります
---— --------------— -----— ----—
返り値 xxxx void 返り値なし

4. setAttachPeriodS

割り込み秒数をセットします. 秒で指定すること

番号 引数名 型 説明
第1引数 attach_period_s double 割り込み周期. 秒で指定
---— --------------— -----— ----—
返り値 xxxx void 返り値なし

5. setMaxOutputValue

出力の最大値をセットします

番号 引数名 型 説明
第1引数 max_output_value double 出力の最大値を指定する
---— --------------— -----— ----—
返り値 xxxx void 返り値なし
